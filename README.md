# Xray Installer

Hardened one-click installer script for:
- VLESS + REALITY (TCP, `xtls-rprx-vision`)
- Shadowsocks 2022 (`2022-blake3-aes-256-gcm`)

Script file: `xray-install.sh`

## One-Click Usage

If this repository is **public**, run:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/Kpowered/xray-install-script/main/xray-install.sh)
```

If this repository is **private**, run:

```bash
git clone https://github.com/Kpowered/xray-install-script.git && cd xray-install-script && sudo bash xray-install.sh
```
